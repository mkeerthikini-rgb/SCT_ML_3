# Cats and Dogs Image Classification using SVM

This project implements a Support Vector Machine (SVM) model to classify images of cats and dogs.

The project was developed using Python in Google Colab and uses image preprocessing techniques for machine learning classification.

## Dataset Used

The Cats vs Dogs dataset was taken from Google Chrome.

Dataset Source:
A ZIP file containing cat and dog images was downloaded from Google Chrome and extracted before using the images for model training.

## Technologies Used

- Python
- OpenCV
- NumPy
- Scikit-learn
- Google Colab
- GitHub

## Machine Learning Algorithm

- Support Vector Machine (SVM)

## Features

- Image upload and preprocessing
- Image resizing
- Feature extraction
- Cat and dog classification
- Accuracy prediction using SVM
